# 飲食策略

> 目標：用自然、可長期維持、能補回平衡的飲食方式，支持代謝健康、腸道菌、血糖穩定與健康長壽。

本文件整理飲食相關策略。重點不是每天填表、禁止所有食物或把生活變成數據工程，而是理解食物如何影響身體，並學會在日常生活中做出比較好的選擇。

## 核心原則

- 優先穩定血糖與胰島素
- 減少精製糖與超加工食品
- 優先蛋白質、低碳蔬菜與好油脂
- 澱粉來源優先選擇抗性澱粉
- 透過發酵食物與膳食纖維支持腸道菌
- 加入富含多酚與植化素的植物性食物
- 不追求完美飲食，而是學會補回平衡
- 用身體回饋判斷飲食是否適合自己

## 飲食不是禁止，而是引導

飲食管理不應該變成壓力。

更好的方向不是：

```text
你不能喝手搖飲。
你不能吃燒烤。
你今天吃錯了。
```

而是：

```text
今天喝了手搖飲，血糖可能會起伏。
等等散步 10 分鐘，讓身體穩一點。
```

```text
今天燒烤吃得很開心。
只是青菜比較少，晚一點補一份蔬菜、菇類、海帶或膳食纖維。
不是禁止燒烤，而是幫身體補回平衡。
```

這是一種生活技能：吃了什麼不是問題，真正重要的是能不能理解身體可能需要什麼，然後做一點修正。

## 小提示：飲食順序與血糖穩定

飲食順序很重要。重點不是把所有醣類都禁止，而是讓醣類不要在空腹狀態下太快進入腸道、太快被吸收，造成血糖快速升高。

- 避免空腹喝果汁，尤其是甜果汁或大量水果汁。
- 空腹喝果汁時，液體糖分可能快速通過胃部進入腸道，使血糖上升速度變快。
- 吃正餐時，可以先吃蔬菜，再吃蛋白質與油脂，最後再吃澱粉或甜的食物。
- 先吃蔬菜、蛋白質與油脂，可以讓胃部開始消化流程，增加胃酸、膽汁與消化液分泌。
- 蛋白質與油脂會讓胃排空速度變慢，類似讓幽門比較收緊，讓食物不要一下子快速進入小腸。
- 這樣可以讓醣類吸收速度變慢，餐後血糖比較穩定，也比較不容易出現快速升高又快速下降的波動。
- 如果真的想喝果汁，優先放在正餐後少量喝，而不是早上起床或空腹時直接喝。
- 比起果汁，完整水果通常更好，因為還保留纖維與咀嚼過程，吸收速度通常比較慢。

可以記成：

```text
先蔬菜 → 再蛋白質與油脂 → 最後醣類
避免空腹果汁 → 降低快速吸收 → 血糖更穩定
```

## 便利商店補缺口策略

不知道吃什麼的時候，便利商店可以是一個實用的補缺口工具。

重點不是去便利商店買完美健康餐，而是用最容易取得的食物，把今天缺的東西補回來。

```text
不知道吃什麼
→ 去便利商店
→ 看今天缺什麼
→ 補蛋白質、補蔬菜、補纖維、補無糖飲料
→ 避免直接用甜飲、炸物、餅乾把一餐解決
```

可以用「缺什麼補什麼」來選：

| 缺口 | 便利商店可選 |
|---|---|
| 蛋白質不足 | 茶葉蛋、水煮蛋、雞胸肉、豆漿、豆腐、鮪魚、毛豆、希臘優格 |
| 蔬菜不足 | 生菜沙拉、海帶、涼拌蔬菜、蔬菜湯、玉米筍、小黃瓜 |
| 膳食纖維不足 | 毛豆、海帶、豆類、沙拉、菇類、膳食纖維補充品 |
| 腸道菌不足 | 無糖優格、泡菜、味噌湯、納豆 |
| 想喝飲料 | 無糖茶、無糖抹茶、黑咖啡、氣泡水 |
| 想吃點心 | 堅果、藍莓、低糖高可可巧克力、無糖優格 |
| 運動後恢復 | 雞胸肉、茶葉蛋、無糖豆漿、高蛋白飲、香蕉 |

便利商店組合範例：

```text
低糖穩定型：
雞胸肉 + 茶葉蛋 + 生菜沙拉 + 無糖茶

腸道菌型：
無糖優格 + 毛豆 + 海帶 + 泡菜

運動恢復型：
高蛋白飲 + 茶葉蛋 + 香蕉 + 無糖豆漿

深夜不想吃太重：
味噌湯 + 豆腐 + 茶葉蛋 + 無糖熱茶
```

便利商店不是完美飲食，但它可以降低決策成本。當不知道吃什麼時，就先用便利商店把身體缺的基本營養補回來。

## 生活技能解鎖

飲食也可以像解鎖技能：

```text
血糖控制技能：
喝甜飲後，知道可以散步、深蹲或延後下一次進食。

纖維補充技能：
吃燒烤、火鍋、外食後，知道補蔬菜、菇類、海帶、豆類或膳食纖維。

蛋白質補足技能：
今天吃得太隨便時，知道晚餐加蛋、魚、豆腐或肉類。

便利商店補缺口技能：
不知道吃什麼時，去便利商店補蛋白質、蔬菜、纖維或無糖飲料。

腸道菌技能：
知道用發酵食物、抗性澱粉、膳食纖維支持腸道環境。

植物多酚技能：
知道用藍莓深色外皮、無糖抹茶、冷壓初榨橄欖油、深色蔬菜與香料，增加多酚與植化素來源。
```

這不是飲食任務表，而是生活技能圖鑑。

## 1. 間歇性斷食 Intermittent Fasting

間歇性斷食的目的不是單純少吃，而是讓身體有較長時間處於低胰島素、低進食刺激狀態。

可能目標：

- 改善胰島素敏感性
- 降低頻繁進食造成的血糖波動
- 讓消化系統有休息時間
- 幫助身體切換到脂肪利用狀態

常見方式：

- 16/8：一天 16 小時不進食，8 小時進食窗口
- 18/6：一天 18 小時不進食，6 小時進食窗口
- OMAD：一天一餐，需謹慎執行

基本原則：

- 斷食期間以水、無糖茶、黑咖啡為主
- 不用斷食來彌補暴食
- 運動量大、睡眠不足、壓力大時，不一定適合拉長斷食
- 身體不適時應停止極端斷食
- 斷食是工具，不是宗教，也不是意志力比賽

身體回饋：

- 精神是否穩定
- 是否過度焦躁或暴食
- 睡眠是否受影響
- 運動恢復是否變差

## 2. 生酮或低碳飲食 Ketogenic / Low Carb

生酮或低碳飲食的核心是降低糖與精製碳水攝取，讓血糖與胰島素波動變小。

可能目標：

- 降低血糖波動
- 降低胰島素需求
- 提升脂肪代謝能力
- 減少對精製澱粉與糖的依賴

飲食方向：

- 優先：蛋、魚、肉、豆製品、低碳蔬菜、好油脂
- 減少：糖、甜飲、麵包、白飯、麵條、甜點、精製澱粉
- 注意電解質：鈉、鉀、鎂
- 注意蛋白質攝取，避免只吃油、不吃足夠蛋白質

需要注意：

- 生酮不等於無限制吃加工肉品
- 長期生酮若有健康風險，需與醫療專業人員討論
- 睡眠不足、壓力過大或運動恢復變差時，不一定適合太極端

身體回饋：

- 精神是否更穩
- 餐後是否比較不想睡
- 運動是否有力
- 情緒是否變得焦躁
- 腸胃是否能接受

## 3. 抗性澱粉 Resistant Starch

本專案目前設定：澱粉來源優先選擇抗性澱粉，而不是精製澱粉。

抗性澱粉是比較不容易在小腸被快速消化吸收的澱粉，會進入大腸被腸道菌發酵，產生短鏈脂肪酸，例如丁酸。

可能好處：

- 降低餐後血糖波動
- 支持腸道菌
- 增加飽足感
- 幫助腸道產生短鏈脂肪酸

常見來源：

- 冷卻後的馬鈴薯
- 冷卻後的白飯或糙米飯
- 冷卻後的地瓜
- 青香蕉或青香蕉粉
- 豆類

實作方式：

- 澱粉煮熟後冷藏，再食用或回溫
- 優先小份量測試身體反應
- 搭配蛋白質、脂肪、蔬菜一起吃
- 不把抗性澱粉當成大量吃澱粉的理由

## 4. 發酵食物 Fermented Foods

發酵食物的目的在於支持腸道菌多樣性與消化功能。

可考慮的食物：

- 無糖優格
- 優酪乳，需注意糖分
- 泡菜
- 德國酸菜
- 味噌
- 納豆
- 康普茶，需注意糖分
- 發酵豆製品

基本原則：

- 從少量開始，觀察腸胃反應
- 優先選擇低糖、少添加物版本
- 發酵食物不是越多越好
- 若有嚴重腸胃疾病、免疫問題，需先諮詢醫師

## 5. 賢者之食卓與腸道菌

賢者之食卓可作為膳食纖維補充策略的一部分，重點放在養腸道菌、增加可發酵纖維來源、幫助餐後血糖管理與改善腸道環境。

本專案將賢者之食卓定位為「益生元 / 水溶性膳食纖維」方向，而不是藥物，也不是取代原型食物的主要營養來源。

可能目標：

- 提供腸道菌可利用的膳食纖維
- 支持短鏈脂肪酸產生，例如丁酸
- 幫助餐後血糖與胰島素反應更平穩
- 增加飽足感
- 輔助排便順暢

使用原則：

- 從低劑量開始，觀察腸胃反應
- 優先搭配正餐使用，而不是拿來抵消高糖高澱粉飲食
- 搭配足夠水分
- 若出現脹氣、腹痛、腹瀉，先減量或暫停
- 與發酵食物、抗性澱粉、低碳蔬菜一起納入腸道菌策略

腸道菌策略：

```text
1. 抗性澱粉：提供腸道菌發酵來源
2. 發酵食物：增加飲食中的微生物與發酵產物
3. 賢者之食卓：補充水溶性膳食纖維 / 益生元
4. 低糖飲食：避免糖分過量干擾代謝與腸道環境
5. 多樣化蔬菜：提供不同種類纖維與植化素
```

## 6. 植物多酚與抗壓適應

植物不是只提供熱量與維生素，也提供多酚、植化素、纖維與不同顏色的保護性成分。

有些植物在缺水、強光、低溫、昆蟲、病原或其他環境壓力下，會產生保護自己的化合物。人吃進這些植物成分，不是直接獲得「抗老保證」，而是可能透過多酚、植化素與 hormesis 類似的輕微刺激，支持身體的抗壓、代謝與修復路徑。

可以理解成：

```text
植物在逆境中產生保護成分
→ 人吃進多酚與植化素
→ 身體可能得到溫和的適應性刺激
→ 配合低糖、運動、睡眠與低發炎生活
→ 支持長期健康策略
```

常見來源：

- 藍莓深色外皮
- 無糖抹茶
- 冷壓初榨橄欖油
- 深色蔬菜
- 香料與辛香料
- 茶類
- 可可含量高、低糖的可可製品

生活做法：

- 用無糖抹茶或無糖茶取代含糖飲料
- 橄欖油搭配蔬菜，而不是拿來抵消高糖飲食
- 藍莓當作低糖水果點心
- 深色蔬菜、香料、茶類輪流吃
- 不需要追求單一超級食物，重點是多樣化、低糖、可長期維持

注意事項：

- 植物多酚不是藥物
- 不是吃了就保證抗老
- 不應用來抵消高糖、高發炎、睡眠不足或缺乏運動
- 有腸胃敏感、腎臟疾病、正在用藥或特殊疾病者，需注意食物與補充品交互作用

## 初版飲食方向

```text
1. 含糖飲料盡量減少，不需要用罪惡感管理
2. 精製糖與超加工食品越少越好
3. 澱粉優先選抗性澱粉來源
4. 每餐優先蛋白質與低碳蔬菜
5. 可用間歇性斷食管理進食窗口
6. 以低碳或生酮作為代謝方向，但不極端硬撐
7. 少量加入發酵食物
8. 使用賢者之食卓作為膳食纖維與益生元補充策略
9. 加入多酚與植化素來源，例如無糖抹茶、藍莓、橄欖油、深色蔬菜與香料
10. 不知道吃什麼時，用便利商店補缺口
11. 飲食不是完美執行，而是知道如何補回平衡
12. 注意飲食順序，先蔬菜、蛋白質與油脂，最後再吃醣類
13. 避免空腹喝果汁，減少血糖快速升高
```

## 身體回饋

不需要每天填 tracking table。只需要對身體反應保持敏感。

可以感受：

- 餐後是否想睡
- 精神是否穩定
- 飢餓感是否失控
- 腸胃是否舒服
- 排便是否順
- 睡眠是否受影響
- 運動是否有力
- 皮膚、關節、發炎感是否比較穩
- 加入多酚與植化素後，飲食是否更自然、低糖、容易維持
- 用便利商店補缺口後，是否比亂吃更穩定

必要時可使用血糖、血壓、體重、腰圍或抽血檢查作為健康參考，但它們是安全工具，不是生活的主人。
