# 飲食策略

> 目標：用可長期執行、可追蹤、可調整的飲食方式，支持代謝健康、腸道菌、血糖穩定與健康長壽。

本文件先集中整理飲食相關策略，後續可再依主題拆成更細的檔案。

## 核心原則

- 穩定血糖與胰島素
- 減少精製糖與超加工食品
- 優先蛋白質、低碳蔬菜與好油脂
- 澱粉來源優先選擇抗性澱粉
- 透過發酵食物與膳食纖維支持腸道菌
- 所有改變都要小步測試、追蹤反應、逐步調整

## 1. 間歇性斷食 Intermittent Fasting

間歇性斷食的目的不是單純少吃，而是讓身體有較長時間處於低胰島素、低進食刺激狀態。

可能目標：

- 改善胰島素敏感性
- 降低頻繁進食造成的血糖波動
- 讓消化系統有休息時間
- 幫助身體切換到脂肪利用狀態

常見實作方式：

- 16/8：一天 16 小時不進食，8 小時進食窗口
- 18/6：一天 18 小時不進食，6 小時進食窗口
- OMAD：一天一餐，需謹慎執行

基本原則：

- 斷食期間以水、無糖茶、黑咖啡為主
- 不用斷食來彌補暴食
- 運動量大、睡眠不足、壓力大時，不一定適合拉長斷食
- 身體不適時應停止極端斷食

## 2. 生酮飲食 Ketogenic Diet

生酮飲食的核心是降低碳水攝取，讓身體主要使用脂肪與酮體作為能量來源。

可能目標：

- 降低血糖波動
- 降低胰島素需求
- 提升脂肪代謝能力
- 減少對精製澱粉與糖的依賴

飲食方向：

- 主要攝取：蛋、魚、肉、豆製品、低碳蔬菜、好油脂
- 避免或減少：糖、甜飲、麵包、白飯、麵條、甜點、精製澱粉
- 注意電解質：鈉、鉀、鎂
- 注意蛋白質攝取，避免只吃油、不吃足夠蛋白質

需要注意：

- 生酮不等於無限制吃加工肉品
- 長期生酮需要追蹤血脂、尿酸、肝腎功能等指標
- 有慢性病或正在服藥者，應與醫療專業人員討論

## 3. 澱粉類只吃抗性澱粉 Resistant Starch

本專案目前設定：澱粉來源優先選擇抗性澱粉，而不是精製澱粉。

抗性澱粉是比較不容易在小腸被快速消化吸收的澱粉，會進入大腸被腸道菌發酵，產生短鏈脂肪酸，例如丁酸。

可能好處：

- 降低餐後血糖波動
- 支持腸道菌
- 增加飽足感
- 幫助腸道產生短鏈脂肪酸

常見來源：

- 冷卻後的馬鈴薯
- 冷卻後的白飯或糙米飯
- 冷卻後的地瓜
- 青香蕉或青香蕉粉
- 豆類

實作方式：

- 澱粉煮熟後冷藏，再食用或回溫
- 優先小份量測試血糖反應
- 搭配蛋白質、脂肪、蔬菜一起吃
- 避免把抗性澱粉當成大量吃澱粉的理由

## 4. 發酵食物 Fermented Foods

發酵食物的目的在於支持腸道菌多樣性與消化功能。

可考慮的食物：

- 無糖優格
- 優酪乳，需注意糖分
- 泡菜
- 德國酸菜
- 味噌
- 納豆
- 康普茶，需注意糖分
- 發酵豆製品

基本原則：

- 從少量開始，觀察腸胃反應
- 優先選擇低糖、少添加物版本
- 發酵食物不是越多越好
- 若有嚴重腸胃疾病、免疫問題，需先諮詢醫師

## 5. 賢者之食卓與腸道菌

賢者之食卓可作為膳食纖維補充策略的一部分，重點放在養腸道菌、增加可發酵纖維來源、幫助餐後血糖管理與改善腸道環境。

本專案將賢者之食卓定位為「益生元 / 水溶性膳食纖維」方向，而不是藥物，也不是取代原型食物的主要營養來源。

可能目標：

- 提供腸道菌可利用的膳食纖維
- 支持短鏈脂肪酸產生，例如丁酸
- 幫助餐後血糖與胰島素反應更平穩
- 增加飽足感
- 輔助排便順暢

使用原則：

- 從低劑量開始，觀察腸胃反應
- 優先搭配正餐使用，而不是拿來抵消高糖高澱粉飲食
- 搭配足夠水分
- 若出現脹氣、腹痛、腹瀉，先減量或暫停
- 與發酵食物、抗性澱粉、低碳蔬菜一起納入腸道菌策略

腸道菌策略：

```text
1. 抗性澱粉：提供腸道菌發酵來源
2. 發酵食物：增加飲食中的微生物與發酵產物
3. 賢者之食卓：補充水溶性膳食纖維 / 益生元
4. 低糖飲食：避免糖分過量干擾代謝與腸道環境
5. 多樣化蔬菜：提供不同種類纖維與植化素
```

## 初版飲食規則

```text
1. 不喝含糖飲料
2. 不吃精製糖
3. 澱粉只選抗性澱粉來源
4. 每餐優先蛋白質與低碳蔬菜
5. 使用間歇性斷食控制進食窗口
6. 以生酮或低碳作為主要代謝方向
7. 每天少量加入發酵食物
8. 使用賢者之食卓作為膳食纖維與益生元補充策略
9. 追蹤血糖、體重、腰圍、睡眠、精神狀態、腸胃反應
```

## 飲食追蹤指標

每日追蹤：

- 起床體重
- 腰圍
- 進食窗口
- 是否有吃發酵食物
- 是否有吃抗性澱粉
- 是否使用賢者之食卓
- 腸胃反應：脹氣、腹痛、排便狀態
- 精神狀態
- 飢餓感

可選追蹤：

- 空腹血糖
- 餐後 1 小時血糖
- 餐後 2 小時血糖
- 血酮
- 血壓

定期檢查：

- HbA1c
- 空腹胰島素
- 三酸甘油脂
- HDL / LDL
- hs-CRP
- 肝功能
- 腎功能
- 尿酸
- 維生素 D
