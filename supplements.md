# 保健食品與補充品

> 目標：整理健康長壽相關的保健食品與補充品，建立「目的明確、劑量保守、安全觀察、避免過度堆疊」的使用框架。

本文件不是醫療建議，也不是推薦所有人都要使用補充品。補充品的定位是輔助飲食、運動、睡眠與疾病風險管理，而不是取代基礎生活方式。

補充品需要比一般飲食更謹慎。這不代表每天都要填 tracking table，而是要知道自己為什麼使用、身體有什麼反應、什麼情況要暫停。

## 核心原則

- 先建立飲食、運動、睡眠與壓力管理，再考慮補充品
- 每次只新增一種補充品，方便判斷身體反應
- 從低劑量開始，觀察 1-2 週
- 不同補充品不要一次大量堆疊
- 有慢性病、正在用藥、手術前後、懷孕或哺乳，需先諮詢醫師
- 補充品要有明確目的、身體回饋與停止條件

## 補充品不是捷徑

補充品不應該拿來抵消不良生活方式。

例如：

```text
高糖、高發炎、睡眠不足、壓力很大
→ 不應該靠 NMN 或其他抗老補充品硬補。

飲食缺蛋白質、蔬菜、纖維
→ 應先補回基本食物結構，而不是只靠膠原蛋白或纖維粉。
```

補充品比較像輔助工具，不是主角。

真正的主角仍然是：

- 睡眠
- 飲食
- 運動
- 壓力調節
- 人際連結
- 身體恢復能力

## 1. NO 一氧化氮支持與左旋精氨酸 L-Arginine

NO（一氧化氮）與血管舒張、血流、運動表現與循環健康有關。本專案把 NO 支持定位為「循環與血管健康」方向。

左旋精氨酸是 NO 生成路徑的前驅物之一，可作為 NO 支持策略的一部分。因此，NO 與左旋精氨酸應視為同一個主題下的不同層次：NO 是目標路徑，左旋精氨酸是可能使用的補充品之一。

### 可能目標

- 支持血管內皮功能
- 支持 NO 路徑
- 改善血流與循環感
- 支持運動耐受度
- 支持運動前血流感
- 支持心血管健康策略

### 生活策略優先

在考慮左旋精氨酸或其他 NO 相關補充品之前，應先建立生活策略。

- 規律 Zone 2 有氧
- 基礎肌力訓練
- 攝取富含硝酸鹽的蔬菜，例如甜菜根、芝麻葉、菠菜
- 控制血糖與慢性發炎
- 避免抽菸
- 減少超加工食品與高糖飲食

### 左旋精氨酸的使用原則

左旋精氨酸不應視為越多越好，也不應拿來取代運動、飲食與血管健康管理。

- 先確認目的：循環、運動前血流、血管健康
- 從低劑量開始測試腸胃與血壓反應
- 不與多種降血壓、血管擴張相關補充品或藥物任意堆疊
- 若有低血壓、心血管疾病、腎臟病、皰疹反覆發作或正在用藥，需先諮詢醫師

### 身體回饋與安全觀察

- 運動耐受度是否改善
- 手腳循環感是否改善
- 是否出現頭暈、心悸或血壓過低感
- 是否出現腸胃不適
- 皰疹發作頻率是否增加
- 若正在使用降血壓藥物或血管擴張相關藥物，需先諮詢醫師

### 停止條件

- 頭暈
- 心悸
- 腸胃不適
- 血壓過低
- 皰疹發作頻率增加

## 2. 鈣片 Calcium

鈣片的目標是支持骨骼健康，但優先順序應該是飲食鈣來源、維生素 D 狀態、肌力訓練、日照與骨骼健康狀態。鈣片不應盲目長期高劑量補充。

### 可能目標

- 支持骨骼健康
- 補足飲食鈣攝取不足
- 搭配維生素 D、蛋白質、肌力訓練與日照一起看

### 使用原則

- 優先從食物取得鈣
- 搭配維生素 D、鎂、蛋白質與阻力訓練一起看
- 若飲食鈣不足，再考慮補充
- 避免高劑量鈣片長期自行使用
- 有腎結石、腎臟病、高血鈣或心血管疾病風險者，需先諮詢醫師

### 身體回饋與安全觀察

- 是否有腸胃不適或便秘
- 是否有腎結石風險
- 骨密度、維生素 D、腎功能等可由醫療檢查確認

## 3. 膠原蛋白 Collagen

膠原蛋白可作為皮膚、關節、肌腱與結締組織支持策略的一部分。它不是肌肉蛋白的主要來源，仍需確保完整蛋白質攝取。

### 可能目標

- 支持關節與肌腱
- 支持皮膚與結締組織
- 搭配運動恢復與關節保護

### 使用原則

- 可搭配維生素 C 攝取，支援膠原合成
- 與肌力訓練、瑜珈、游泳、關節保護一起看
- 不取代蛋、魚、肉、豆製品等完整蛋白質
- 若目標是關節與肌腱，需配合適當負荷與恢復

### 身體回饋

- 關節是否比較舒服
- 肌腱恢復感是否改善
- 運動後不適感是否降低
- 是否有腸胃不適

## 4. 乾燥啤酒酵母 EBIOS / Brewer's Yeast

EBIOS（エビオス錠）屬於乾燥啤酒酵母類補充品。它不是強效益生菌，也不是抗老化特效補充品，而是偏向「胃腸營養補給」與「基礎營養補缺口」的保健食品。

乾燥啤酒酵母通常含有 B 群相關營養素、胺基酸、礦物質與膳食纖維，可作為飲食不規律、胃口較差、腸胃狀態不穩或營養攝取不足時的輔助工具。

### 可能目標

- 支持胃腸營養補給
- 補充 B 群相關營養素
- 補充胺基酸與礦物質
- 補充少量膳食纖維
- 在飲食不規律、胃口差或消化狀態不佳時，作為基礎營養補缺口

### 使用定位

EBIOS 應定位為「補缺口」而不是「治療疾病」。

```text
熬夜、飲食亂、胃口差、營養不足
→ 補一點基礎營養
→ 支持胃腸與恢復
```

它可以放在腸胃支持與飲食補缺口策略中一起看，但不應取代正餐、蛋白質、蔬菜、纖維與睡眠。

### 使用原則

- 依產品標示使用，不自行超量
- 初次使用可從較低量開始，觀察腸胃反應
- 不把它當作取代正餐或治療腸胃疾病的工具
- 若有痛風、高尿酸、腎臟病或正在限制普林攝取，需先諮詢醫師
- 若有嚴重腸胃症狀、持續腹痛、黑便、體重異常下降或長期食慾不振，應就醫確認原因

### 身體回饋與安全觀察

- 胃脹、消化感是否改善
- 排便是否變順或變差
- 是否出現脹氣、放屁增加或腸胃不適
- 食慾是否明顯改變
- 尿酸或痛風狀態是否受到影響

### 停止條件

- 明顯腸胃不適
- 脹氣或腹瀉加重
- 痛風發作或尿酸控制變差
- 皮膚過敏、搔癢或其他不適反應
- 任何不明原因的不舒服

## 5. NMN 與 DNA 保護

NMN（Nicotinamide Mononucleotide）常被放在 NAD+、細胞能量、粒線體功能與 DNA 修復相關策略中討論。本專案將 NMN 定位為「在身體低發炎、代謝穩定的前提下，支持細胞修復與 DNA 保護的補充品候選」。

重點不是單靠 NMN 逆轉老化，而是先讓身體處於比較適合修復的狀態：低慢性發炎、血糖穩定、睡眠足夠、運動適量、腸道環境良好。

若身體長期處於高糖、高發炎、睡眠不足、過度壓力狀態，不建議補充 NMN；應先暫停 NMN，優先修正血糖、發炎、睡眠與壓力問題，等身體回到較穩定、低發炎、可修復的狀態後，再重新評估是否需要使用。

### 可能目標

- 支持 NAD+ 相關代謝路徑
- 支持細胞能量與粒線體功能
- 支持 DNA 修復與細胞壓力反應
- 作為健康長壽策略中的輔助工具

### 前提條件

- 血糖與胰島素相對穩定
- 慢性發炎狀態不明顯
- 睡眠品質穩定
- 運動量適中，不過度訓練
- 飲食以低糖、低加工、足夠蛋白質與蔬菜為主
- 腸道菌策略已建立，例如發酵食物、抗性澱粉、膳食纖維

### 不建議或應暫停 NMN 的狀態

```text
1. 長期高糖飲食或血糖控制不佳
2. 慢性發炎明顯
3. 睡眠不足或睡眠品質長期不穩定
4. 長期過度壓力、焦慮、疲勞或恢復不良
5. 過度訓練、心率偏高、恢復變差
6. 正在生病、感染、發燒或身體處於急性壓力狀態
7. 未釐清的癌症風險、癌症治療中或醫師不建議使用
```

### 使用原則

- 不把 NMN 當作抵消不良生活習慣的工具
- 先建立飲食、運動、睡眠與發炎管理
- 從低劑量開始，觀察精神、睡眠、腸胃與心率反應
- 避免同時與太多抗老補充品堆疊，否則難以判斷效果
- 若有癌症病史、正在治療癌症、肝腎疾病、懷孕、哺乳或正在使用藥物，需先諮詢醫師

### 停止條件

- 睡眠變差
- 心悸或心率異常
- 腸胃不適
- 焦躁或精神過度亢奮
- 肝腎指標異常
- 血糖、發炎、睡眠或壓力狀態變差

## 安全觀察模板

補充品可以用簡單方式觀察，不需要把生活變成表格工程。

```markdown
# 補充品安全觀察

- 名稱：
- 目的：為什麼要用？
- 劑量：從低劑量開始
- 使用時間：

## 身體回饋

- 睡眠是否變好或變差：
- 腸胃是否舒服：
- 心率、血壓或循環感是否異常：
- 運動恢復是否有變化：
- 皮膚、關節、肌腱是否有感：
- 是否有任何不適：

## 決定

- 保留 / 降低劑量 / 暫停 / 停止
```

## 初版補充品方向

```text
1. 補充品不是越多越好
2. 每次只新增一種
3. 從低劑量開始
4. 至少觀察 1-2 週
5. 有不適先暫停
6. 有慢性病或用藥先問醫師
7. 補充品不能取代飲食、運動、睡眠
8. 用身體回饋與安全狀態決定是否保留
9. NMN 不應用來抵消高糖、高發炎、睡眠不足與過度壓力狀態
```
