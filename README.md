# 150-year-old

> 目標：系統化整理「如何活得更健康、更長壽」的知識，建立一套可追蹤、可實驗、可調整的健康長壽生活方式。

本專案名稱 `150-year-old` 代表一個長期健康目標與研究方向，不代表保證可以活到 150 歲。這是一個以公開資料、YouTube 內容整理、個人實踐紀錄與健康科學筆記為核心的知識庫。

## 核心理念

健康長壽不是單一方法，而是多個系統一起維持：

- 穩定血糖與胰島素
- 降低慢性發炎
- 維持粒線體功能
- 保護肌肉量與骨密度
- 維持腸道菌多樣性
- 改善睡眠與壓力反應
- 減少過度加工食品
- 透過週期性壓力刺激提升適應力

## 飲食策略

目前飲食方向以以下幾個主軸為基礎。

### 1. 間歇性斷食 Intermittent Fasting

間歇性斷食的目的不是單純少吃，而是讓身體有較長時間處於低胰島素、低進食刺激狀態。

可能目標：

- 改善胰島素敏感性
- 降低頻繁進食造成的血糖波動
- 讓消化系統有休息時間
- 幫助身體切換到脂肪利用狀態

常見實作方式：

- 16/8：一天 16 小時不進食，8 小時進食窗口
- 18/6：一天 18 小時不進食，6 小時進食窗口
- OMAD：一天一餐，需謹慎執行

基本原則：

- 斷食期間以水、無糖茶、黑咖啡為主
- 不用斷食來彌補暴食
- 運動量大、睡眠不足、壓力大時，不一定適合拉長斷食
- 身體不適時應停止極端斷食

### 2. 生酮飲食 Ketogenic Diet

生酮飲食的核心是降低碳水攝取，讓身體主要使用脂肪與酮體作為能量來源。

可能目標：

- 降低血糖波動
- 降低胰島素需求
- 提升脂肪代謝能力
- 減少對精製澱粉與糖的依賴

飲食方向：

- 主要攝取：蛋、魚、肉、豆製品、低碳蔬菜、好油脂
- 避免或減少：糖、甜飲、麵包、白飯、麵條、甜點、精製澱粉
- 注意電解質：鈉、鉀、鎂
- 注意蛋白質攝取，避免只吃油、不吃足夠蛋白質

需要注意：

- 生酮不等於無限制吃加工肉品
- 長期生酮需要追蹤血脂、尿酸、肝腎功能等指標
- 有慢性病或正在服藥者，應與醫療專業人員討論

### 3. 澱粉類只吃抗性澱粉 Resistant Starch

本專案目前設定：澱粉來源優先選擇抗性澱粉，而不是精製澱粉。

抗性澱粉是比較不容易在小腸被快速消化吸收的澱粉，會進入大腸被腸道菌發酵，產生短鏈脂肪酸，例如丁酸。

可能好處：

- 降低餐後血糖波動
- 支持腸道菌
- 增加飽足感
- 幫助腸道產生短鏈脂肪酸

常見來源：

- 冷卻後的馬鈴薯
- 冷卻後的白飯或糙米飯
- 冷卻後的地瓜
- 青香蕉或青香蕉粉
- 豆類

實作方式：

- 澱粉煮熟後冷藏，再食用或回溫
- 優先小份量測試血糖反應
- 搭配蛋白質、脂肪、蔬菜一起吃
- 避免把抗性澱粉當成大量吃澱粉的理由

### 4. 發酵食物 Fermented Foods

發酵食物的目的在於支持腸道菌多樣性與消化功能。

可考慮的食物：

- 無糖優格
- 優酪乳，需注意糖分
- 泡菜
- 德國酸菜
- 味噌
- 納豆
- 康普茶，需注意糖分
- 發酵豆製品

基本原則：

- 從少量開始，觀察腸胃反應
- 優先選擇低糖、少添加物版本
- 發酵食物不是越多越好
- 若有嚴重腸胃疾病、免疫問題，需先諮詢醫師

## 初版飲食規則

```text
1. 不喝含糖飲料
2. 不吃精製糖
3. 澱粉只選抗性澱粉來源
4. 每餐優先蛋白質與低碳蔬菜
5. 使用間歇性斷食控制進食窗口
6. 以生酮或低碳作為主要代謝方向
7. 每天少量加入發酵食物
8. 追蹤血糖、體重、腰圍、睡眠、精神狀態
```

## 運動策略

運動的目標以「健康長壽」為主，而不是追求極限表現。原則是能長期維持、降低受傷風險、保護關節、維持肌肉量、提升心肺功能，並避免過度訓練造成過多壓力與自由基生成。

### 運動核心原則

- 健康優先，不以破紀錄或極限訓練為目標
- 選擇低衝擊、低受傷風險、可長期維持的運動
- 運動後應該恢復精神，而不是長期疲勞
- 避免頻繁高強度訓練導致過度氧化壓力
- 重視暖身、收操、呼吸與動作品質
- 逐步增加強度，不突然加量
- 疼痛時停止，不硬撐

### 1. 游泳 Swimming

游泳是適合健康長壽的低衝擊運動，對膝蓋、髖關節與脊椎壓力較小，同時可以訓練心肺、肌耐力與呼吸節奏。

可能好處：

- 提升心肺功能
- 降低關節衝擊
- 訓練全身肌群
- 改善呼吸控制
- 適合體重較重或關節不適者

注意事項：

- 初期以輕鬆游、穩定呼吸為主
- 避免一開始就追求速度與距離
- 肩膀不適時減少自由式划手量
- 重視技術，避免錯誤姿勢造成肩夾擠
- 游泳後補充水分與電解質

### 2. 瑜珈 Yoga

瑜珈的目標是改善柔軟度、活動度、呼吸、平衡與身體覺察，而不是追求高難度姿勢。

可能好處：

- 改善關節活動度
- 放鬆筋膜與肌肉張力
- 改善姿勢控制
- 訓練平衡感
- 降低壓力反應
- 幫助睡眠與副交感神經啟動

注意事項：

- 不追求極限拉伸
- 避免彈震式拉筋
- 膝蓋、下背、脖子有壓力時立即調整
- 以呼吸穩定、可控制的範圍為主
- 高難度倒立、深後彎、深扭轉需謹慎

### 3. 低強度有氧 Zone 2

低強度有氧適合長期建立代謝健康。強度大約是可以講話、但無法輕鬆唱歌的程度。

可選方式：

- 快走
- 游泳
- 腳踏車
- 橢圓機
- 輕鬆慢跑，若關節可承受

基本原則：

- 每週 2-4 次
- 每次 20-45 分鐘
- 避免每次都練到喘爆
- 以穩定、可恢復為主

### 4. 肌力與肌肉量維持

長壽不只需要心肺，也需要肌肉量。肌肉可以幫助血糖控制、保護關節、維持基礎代謝與降低老年跌倒風險。

方向：

- 深蹲或坐站訓練
- 髖關節鉸鏈動作
- 推：伏地挺身、啞鈴推舉
- 拉：划船、彈力帶拉背
- 核心：棒式、死蟲、鳥狗

基本原則：

- 每週 2 次即可起步
- 保留 2-3 下餘力，不每組做到力竭
- 優先動作品質，再增加重量
- 避免突然高重量、高次數、高疲勞

## 避免運動傷害

運動傷害會破壞長期持續性，因此避免受傷比短期進步更重要。

```text
1. 每次運動先暖身 5-10 分鐘
2. 強度只逐步增加，不突然加量
3. 不在睡眠不足、壓力大、身體不適時做高強度訓練
4. 關節痛、刺痛、麻木時停止
5. 運動後做輕度收操與伸展
6. 每週安排低強度日或休息日
7. 技術不熟的動作先降低重量與速度
8. 以能持續 10 年的運動方式為優先
```

## 減少自由基與過度氧化壓力

運動本身會產生一定程度的氧化壓力，適量時可以促進身體適應；但過度高強度、恢復不足、長期疲勞，可能讓壓力大於恢復。

目標不是完全避免自由基，而是避免「過量且無法恢復」的運動壓力。

實作原則：

- 以中低強度運動作為主體
- 高強度訓練少量即可，不頻繁堆疊
- 運動後重視睡眠、蛋白質、電解質與水分
- 避免空腹狀態下進行過長或過強訓練
- 若出現長期疲勞、睡眠變差、心率偏高，降低訓練量
- 飲食中加入足夠蔬菜、發酵食物與優質蛋白質

## 初版運動規則

```text
1. 運動以健康長壽為主，不追求極限
2. 優先選擇游泳、瑜珈、快走、低強度有氧
3. 每週加入 2 次基礎肌力訓練
4. 每次運動保留餘力，不長期練到力竭
5. 關節痛就停止或調整動作
6. 高強度訓練少量即可
7. 睡眠不足時不做高強度訓練
8. 運動後以恢復品質作為判斷標準
```

## 個人實驗原則

所有健康策略都應該以「小步測試、追蹤反應、逐步調整」為原則。

建議流程：

1. 先定義假設
2. 一次只改一個主要變因
3. 設定觀察時間，例如 2 週或 4 週
4. 記錄客觀數據與主觀感受
5. 根據結果調整

範例：

```markdown
# 實驗：16/8 間歇性斷食
```

## 醫療與運動安全免責說明

本專案內容只作為健康知識整理與個人實驗紀錄，不構成醫療建議、診斷、治療或個人化運動處方。若有糖尿病、高血壓、心血管疾病、腎臟病、肝臟疾病、痛風、飲食障礙、懷孕、哺乳，或正在使用降血糖、降血壓、抗凝血等藥物，執行間歇性斷食、生酮飲食或大幅改變飲食前，應先諮詢合格醫療專業人員。

若有心臟病史、胸痛、暈眩、呼吸困難、關節傷病、脊椎問題、術後恢復或長期未運動，開始運動計畫前也應先諮詢醫師或合格運動專業人員。
