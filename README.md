# 150-year-old

> 目標：系統化整理「如何活得更健康、更長壽」的知識，建立一套自然、可長期融入生活的健康長壽方式。

本專案名稱 `150-year-old` 代表一個長期健康目標與研究方向，不代表保證可以活到 150 歲。這是一個以公開資料、YouTube 內容整理、個人實踐、生命科學資料收集、健康科技想像與健康科學筆記為核心的知識庫。

這個專案不是要把人變成機器人，也不是要用軍隊式紀律管理生活。健康應該回到自然、穩定、可持續的日常：吃得簡單、睡得穩、動得舒服、壓力有出口，讓身體在自然節奏中維持修復能力。

同時，本專案也探索 AI 如何成為健康陪伴：不是監控、不是禁止、不是責備，而是透過手機、智慧型手錶、AI glass、avatar 與 AI 對話，幫助人更理解身體、補回平衡、解鎖生活技能。

## 健康生活哲學

不需要 tracking table。

我不認為健康生活一定需要每天填表、打勾、量化所有事情，也不想把自己活成一個機器人，或像軍隊一樣被規則管理。健康長壽的目標不是把生活變成任務清單，而是讓健康選擇變成自然的生活方式。

真正重要的是：

- 日常飲食自然偏向低糖、少加工、足夠蛋白質與蔬菜
- 運動是舒服、可持續、低受傷風險的活動，而不是懲罰
- 睡眠是生活節奏的一部分，不是被數據逼迫
- 補充品是輔助，不是替代生活方式
- 生命科學用來理解方向，不是製造焦慮
- AI 可以是陪伴與引導，不是監控與控制
- 人與人之間的連結也是健康的一部分
- 身體感受比表格更重要
- 能長期自然做到，才是真正有價值的健康策略

這個專案會整理知識、原則與關聯，但不以建立追蹤表為目標。必要時可以觀察身體變化，例如精神、睡眠、體力、腸胃反應、疼痛與恢復感，但不需要把生活變成數據工程。

## 核心理念

健康長壽不是單一方法，而是多個系統一起維持：

- 穩定血糖與胰島素
- 降低慢性發炎
- 維持粒線體功能
- 保護肌肉量與骨密度
- 維持腸道菌多樣性
- 改善睡眠與壓力反應
- 減少過度加工食品
- 透過適度、短暫、可恢復的壓力刺激提升適應力
- 關注重大疾病風險，例如糖尿病、癌症、失智症、心血管疾病
- 區分已較穩定的生命科學基礎與仍需未來研究的前沿方向
- 重視安全感、情緒出口、社群與人際連結
- 透過 AI avatar、AI glass、智慧型手錶與手機，把健康引導自然融入生活
- 把健康行為變成生活技能解鎖，而不是任務壓力

## 個人實踐原則

所有健康策略都應該以「自然融入生活、身體感受良好、能長期維持」為原則。

建議方向：

1. 不追求完美執行
2. 不需要每天填 tracking table
3. 一次只調整一個主要生活習慣
4. 用身體感受判斷是否適合
5. 若造成焦慮、壓力或失去生活感，就需要調整
6. 能自然維持，比短期極端更重要
7. AI 可以協助提醒與陪伴，但不應成為新的壓力來源
8. 健康行為可以變成生活技能解鎖，而不是 KPI
