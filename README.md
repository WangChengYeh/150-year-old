# 150-year-old

> 目標：系統化整理「如何活得更健康、更長壽」的知識，建立一套可追蹤、可實驗、可調整的健康長壽生活方式。

本專案名稱 `150-year-old` 代表一個長期健康目標與研究方向，不代表保證可以活到 150 歲。這是一個以公開資料、YouTube 內容整理、個人實踐紀錄與健康科學筆記為核心的知識庫。

## 核心理念

健康長壽不是單一方法，而是多個系統一起維持：

- 穩定血糖與胰島素
- 降低慢性發炎
- 維持粒線體功能
- 保護肌肉量與骨密度
- 維持腸道菌多樣性
- 改善睡眠與壓力反應
- 減少過度加工食品
- 透過週期性壓力刺激提升適應力

## 飲食策略

目前飲食方向以以下幾個主軸為基礎。

### 1. 間歇性斷食 Intermittent Fasting

間歇性斷食的目的不是單純少吃，而是讓身體有較長時間處於低胰島素、低進食刺激狀態。

可能目標：

- 改善胰島素敏感性
- 降低頻繁進食造成的血糖波動
- 讓消化系統有休息時間
- 幫助身體切換到脂肪利用狀態

常見實作方式：

- 16/8：一天 16 小時不進食，8 小時進食窗口
- 18/6：一天 18 小時不進食，6 小時進食窗口
- OMAD：一天一餐，需謹慎執行

基本原則：

- 斷食期間以水、無糖茶、黑咖啡為主
- 不用斷食來彌補暴食
- 運動量大、睡眠不足、壓力大時，不一定適合拉長斷食
- 身體不適時應停止極端斷食

### 2. 生酮飲食 Ketogenic Diet

生酮飲食的核心是降低碳水攝取，讓身體主要使用脂肪與酮體作為能量來源。

可能目標：

- 降低血糖波動
- 降低胰島素需求
- 提升脂肪代謝能力
- 減少對精製澱粉與糖的依賴

飲食方向：

- 主要攝取：蛋、魚、肉、豆製品、低碳蔬菜、好油脂
- 避免或減少：糖、甜飲、麵包、白飯、麵條、甜點、精製澱粉
- 注意電解質：鈉、鉀、鎂
- 注意蛋白質攝取，避免只吃油、不吃足夠蛋白質

需要注意：

- 生酮不等於無限制吃加工肉品
- 長期生酮需要追蹤血脂、尿酸、肝腎功能等指標
- 有慢性病或正在服藥者，應與醫療專業人員討論

### 3. 澱粉類只吃抗性澱粉 Resistant Starch

本專案目前設定：澱粉來源優先選擇抗性澱粉，而不是精製澱粉。

抗性澱粉是比較不容易在小腸被快速消化吸收的澱粉，會進入大腸被腸道菌發酵，產生短鏈脂肪酸，例如丁酸。

可能好處：

- 降低餐後血糖波動
- 支持腸道菌
- 增加飽足感
- 幫助腸道產生短鏈脂肪酸

常見來源：

- 冷卻後的馬鈴薯
- 冷卻後的白飯或糙米飯
- 冷卻後的地瓜
- 青香蕉或青香蕉粉
- 豆類

實作方式：

- 澱粉煮熟後冷藏，再食用或回溫
- 優先小份量測試血糖反應
- 搭配蛋白質、脂肪、蔬菜一起吃
- 避免把抗性澱粉當成大量吃澱粉的理由

### 4. 發酵食物 Fermented Foods

發酵食物的目的在於支持腸道菌多樣性與消化功能。

可考慮的食物：

- 無糖優格
- 優酪乳，需注意糖分
- 泡菜
- 德國酸菜
- 味噌
- 納豆
- 康普茶，需注意糖分
- 發酵豆製品

基本原則：

- 從少量開始，觀察腸胃反應
- 優先選擇低糖、少添加物版本
- 發酵食物不是越多越好
- 若有嚴重腸胃疾病、免疫問題，需先諮詢醫師

## 初版飲食規則

```text
1. 不喝含糖飲料
2. 不吃精製糖
3. 澱粉只選抗性澱粉來源
4. 每餐優先蛋白質與低碳蔬菜
5. 使用間歇性斷食控制進食窗口
6. 以生酮或低碳作為主要代謝方向
7. 每天少量加入發酵食物
8. 追蹤血糖、體重、腰圍、睡眠、精神狀態
```

## 建議追蹤指標

### 每日追蹤

- 起床體重
- 腰圍
- 睡眠時間
- 精神狀態
- 飢餓感
- 進食窗口
- 是否有吃發酵食物
- 是否有吃抗性澱粉

### 可選追蹤

- 空腹血糖
- 餐後 1 小時血糖
- 餐後 2 小時血糖
- 血酮
- 血壓
- 靜息心率
- HRV

### 定期健康檢查

- HbA1c
- 空腹胰島素
- 三酸甘油脂
- HDL / LDL
- hs-CRP
- 肝功能
- 腎功能
- 尿酸
- 維生素 D

## 個人實驗原則

所有健康策略都應該以「小步測試、追蹤反應、逐步調整」為原則。

建議流程：

1. 先定義假設
2. 一次只改一個主要變因
3. 設定觀察時間，例如 2 週或 4 週
4. 記錄客觀數據與主觀感受
5. 根據結果調整

範例：

```markdown
# 實驗：16/8 間歇性斷食

每天 16 小時不進食，可以改善精神穩定度與降低體重。
```
