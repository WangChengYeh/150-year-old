# 150-year-old

> 目標：系統化整理「如何活得更健康、更長壽」的知識，建立一套可追蹤、可實驗、可調整的健康長壽生活方式。

本專案名稱 `150-year-old` 代表一個長期健康目標與研究方向，不代表保證可以活到 150 歲。這是一個以公開資料、YouTube 內容整理、個人實踐紀錄、生命科學研究追蹤與健康科學筆記為核心的知識庫。

## 核心理念

健康長壽不是單一方法，而是多個系統一起維持：

- 穩定血糖與胰島素
- 降低慢性發炎
- 維持粒線體功能
- 保護肌肉量與骨密度
- 維持腸道菌多樣性
- 改善睡眠與壓力反應
- 減少過度加工食品
- 透過週期性壓力刺激提升適應力
- 追蹤重大疾病風險，例如糖尿病、癌症、失智症、心血管疾病
- 追蹤幹細胞、癌細胞 mRNA、DNA 修復、細胞老化與再生醫學等生命科學方向

## 個人實驗原則

所有健康策略都應該以「小步測試、追蹤反應、逐步調整」為原則。

建議流程：

1. 先定義假設
2. 一次只改一個主要變因
3. 設定觀察時間，例如 2 週或 4 週
4. 記錄客觀數據與主觀感受
5. 根據結果調整
