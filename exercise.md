# 運動策略

> 目標：以健康長壽為核心，建立低受傷風險、可長期維持、能支持代謝健康、心肺功能、肌肉量與活動度的運動系統。

本文件集中整理健康長壽取向的運動策略，而不是競技運動或極限訓練。

## 核心分類

運動壓力是大分類，可以分成三個主要方向：

```text
運動
├── 有氧運動
├── 抗阻力訓練
└── 活動度 / 瑜珈 / 伸展
```

三者目標不同，但都應該以「短暫、適量、可恢復」為原則。

| 類型 | 主要目標 |
|---|---|
| 有氧運動 | 心肺功能、粒線體、血糖利用、循環與大腦血流 |
| 抗阻力訓練 | 肌肉量、骨密度、關節穩定、胰島素敏感性、日常活動能力 |
| 活動度 / 瑜珈 / 伸展 | 關節活動度、平衡、神經控制、呼吸、放鬆與壓力調節 |

## 核心原則

- 健康優先，不追求極限表現
- 選擇低受傷風險、可長期維持的運動
- 同時維持心肺功能、肌肉量、活動度與平衡感
- 避免過度訓練與慢性疲勞
- 運動後應該恢復精神，而不是長期耗損
- 重視暖身、收操、呼吸與動作品質
- 運動最好是自願的、被吸引的，而不是被強迫的
- 刺激不是目的，適應才是目的，恢復是適應的前提

## 1. 有氧運動

有氧運動主要訓練心肺功能、粒線體、血糖利用能力、循環系統與大腦血流。

可選方式：

- 快走
- 游泳
- 腳踏車
- 橢圓機
- Zone 2 低強度有氧
- 輕鬆慢跑，若關節可承受

### Zone 2 低強度有氧

低強度有氧適合長期建立代謝健康。強度大約是可以講話、但無法輕鬆唱歌的程度。

基本原則：

- 每週 2-4 次
- 每次 20-45 分鐘
- 避免每次都練到喘爆
- 以穩定、可恢復為主

### 游泳 Swimming

游泳是適合健康長壽的低衝擊有氧運動，對膝蓋、髖關節與脊椎壓力較小，同時可以訓練心肺、肌耐力與呼吸節奏。

可能好處：

- 提升心肺功能
- 降低關節衝擊
- 訓練全身肌群
- 改善呼吸控制
- 適合體重較重或關節不適者

注意事項：

- 初期以輕鬆游、穩定呼吸為主
- 避免一開始就追求速度與距離
- 肩膀不適時減少自由式划手量
- 重視技術，避免錯誤姿勢造成肩夾擠
- 游泳後補充水分與電解質

## 2. 抗阻力訓練

抗阻力訓練是運動壓力中，專門刺激肌肉、骨骼與代謝適應的重要類型。

長壽不只需要心肺，也需要肌肉量。肌肉可以幫助血糖控制、保護關節、維持基礎代謝與降低老年跌倒風險。

```text
抗阻力訓練
→ 肌肉與骨骼受到可控挑戰
→ 身體啟動修復與適應
→ 肌肉、骨骼、代謝與日常功能變強
```

可能好處：

- 維持或增加肌肉量
- 提升肌力
- 支持骨密度
- 提升關節穩定性
- 改善胰島素敏感性
- 提升日常活動能力
- 降低跌倒風險

基本原則：

- 每週 2-3 次即可起步
- 每次選 2-4 個動作
- 每個動作 2-3 組
- 每組保留 2-3 下餘力
- 不需要每次做到力竭
- 動作品質優先於重量與次數
- 肌肉痠可以接受，關節痛不應硬撐

### 小提示：休息與鍛鍊一樣重要

抗阻力訓練不是只看做了多少下、拿了多重。真正讓身體變強的，不是訓練當下，而是訓練後的修復與適應。

```text
訓練
→ 給肌肉、骨骼與神經一個刺激
→ 休息與營養
→ 身體修復
→ 下次變得更穩、更有力
```

休息與鍛鍊一樣重要。身體是在休息的時候變強，不是在硬撐的時候變強。

可以用保守的節奏理解恢復：

- 上半身訓練後，至少隔一天再訓練同一批肌群。
- 下半身訓練後，恢復需求通常更高，可以隔兩天再訓練同一批肌群。
- 若肌肉痠痛、關節不舒服、睡眠變差或精神疲勞，就要再多休息。
- 不需要每天訓練同一部位，也不需要每次都做到力竭。

目標不是變成健美先生或健美小姐，而是建立健康、穩定、可長期使用的身體。

過度運動會讓壓力大於恢復，反而可能增加疲勞、發炎、受傷與衰老感。抗阻力訓練應該讓身體更有支撐感，而不是把身體耗乾。

## 3. 活動度 / 瑜珈 / 伸展

活動度、瑜珈與伸展主要訓練關節活動範圍、平衡、神經控制、呼吸、身體覺察與壓力調節。

目標不是追求高難度姿勢，而是讓身體更能動、更舒服、更穩定。

可能好處：

- 改善關節活動度
- 放鬆筋膜與肌肉張力
- 改善姿勢控制
- 訓練平衡感
- 降低壓力反應
- 幫助睡眠與副交感神經啟動

注意事項：

- 不追求極限拉伸
- 避免彈震式拉筋
- 膝蓋、下背、脖子有壓力時立即調整
- 以呼吸穩定、可控制的範圍為主
- 高難度倒立、深後彎、深扭轉需謹慎

### 小提示

這裡記錄一些居家瑜珈的重要發現與心得，重點不是增加強度，而是創造容易開始、容易放鬆、容易持續的環境。

- 居家瑜珈可以安排在光線偏暗的環境中，使用偏黃色或偏紅色的燈光，讓身體比較容易放鬆。
- 可以在床上做輕柔瑜珈，但動作要以安全、穩定、低強度為主，避免需要大量平衡或支撐的動作。
- 穿著柔軟、舒適、貼身但不緊繃的衣物，例如居家內衣型或瑜珈服，有助於降低身體束縛感。
- 睡前可以做放鬆型瑜珈，作為幫助入睡的睡前儀式。
- 失眠時可以改做緩慢、低刺激的瑜珈與伸展，讓注意力回到呼吸與身體感受。
- 早起也可以做簡短瑜珈，作為喚醒身體、啟動一天的方式。
- 使用瑜珈球、瑜珈磚、經脈球等輔具，可以讓居家瑜珈更有趣，也增加身體探索感。
- 瑜珈球可以增加不穩定性，讓核心、平衡與控制力受到更多刺激，但動作要從簡單、安全的版本開始。
- 瑜珈磚可以降低動作門檻，幫助身體在不到位時仍維持穩定與放鬆，也可以讓伸展更容易持續。
- 經脈球可以用來按摩足底、臀部、背部或緊繃部位，讓瑜珈前後的放鬆更有感。
- 輔具可以讓鍛鍊力度加倍，但不應該變成硬撐或追求高難度；感覺有趣、穩定、可恢復才是重點。

## 初版運動規則

```text
1. 運動以健康長壽為主，不追求極限
2. 運動分成有氧運動、抗阻力訓練、活動度 / 瑜珈 / 伸展
3. 有氧運動以快走、游泳、Zone 2 為主
4. 抗阻力訓練以單槓、啞鈴、深蹲、棒式為核心
5. 抗阻力訓練要重視休息，身體是在休息時修復並變強
6. 活動度訓練以瑜珈、伸展、關節控制與呼吸為主
7. 每次運動保留餘力，不長期練到力竭
8. 關節痛就停止或調整動作
9. 高強度訓練少量即可
10. 睡眠不足時不做高強度訓練
11. 運動後以恢復品質作為判斷標準
12. 讓運動變成自願與美好想像，而不是強迫與懲罰
13. 過度運動會讓壓力大於恢復，反而可能增加疲勞、發炎、受傷與衰老感
```

## 身體回饋

不需要 tracking table。只需要感受運動是否讓身體更健康、更穩、更舒服。

可以感受：

- 運動後是否精神更好
- 睡眠是否變穩
- 關節是否舒服
- 身體是否更有支撐感
- 心肺是否更穩
- 肌肉是否有被適度刺激，而不是關節疼痛
- 活動度是否更好
- 情緒是否更輕盈
- 是否更願意自然地繼續動
- 隔天是否恢復，而不是持續疲勞
- 是否需要增加休息，而不是增加訓練量
